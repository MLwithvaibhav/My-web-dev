HTML media player

This HTML Media Player is a simple, browser-based video and audio player built using the video and audio HTML5 elements. It allows users to play, pause, and control the playback of local or online media files directly within the web page, without requiring any external plugins. The player includes customizable controls such as play/pause buttons, volume adjustment, progress bar, and fullscreen toggle. It supports popular media formats like MP4, WEBM, OGG for video and MP3, WAV, OGG for audio. This project demonstrates the integration of multimedia content in web pages, and can be expanded with JavaScript to add advanced features like playlists, speed control, and subtitles.
